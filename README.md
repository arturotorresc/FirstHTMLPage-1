<h1>
    This repository's purpose is to learn how to setup a new git repo
</h1>

<ol>
    <li>
        git init
    </li>
    <li>
        git add file.ext
    </li>
    <li>
        git commit -m "Message"
    </li>
    <li>
        git remote add origin url-to-github-repo
    </li>
    <li>
        git push -u origin master
    </li>
</ol>

<ol>
    <li>
        Fork repository
    </li>
    <li>
        Clone the forked repo
    </li>
    <li>
        make changes
    </li>
    <li>
        git add, git commit -m "message"
    </li>
    <li>
        git push
    </li>
    </li>
    <li>
        pull request
    </li>
</ol>
